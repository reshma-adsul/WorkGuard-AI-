Capabilities

Enterprise-grade
security. Zero friction.

Six layers of intelligent monitoring that work silently in the background — no configuration required.

⌨️
Keystroke Dynamics

Captures dwell time and flight time with millisecond precision. Every person's typing rhythm is as unique as a fingerprint.

🧬
Biometric Fingerprinting

Z-score similarity scoring builds an immutable behavioral profile. Threshold 0.65 separates owner from intruder.

🤖
ML Anomaly Detection

Isolation Forest + One-Class SVM ensemble. Unsupervised learning — no labeled attack data needed.

📷
Camera Surveillance

OpenCV-powered 3-state FSM: Owner Present → No One → Stranger. Smoothing buffer prevents false alerts.

🔐
AES-256-GCM Encryption

Every log chunk encrypted with a unique 96-bit nonce. GCM authentication tag detects tampered files.

📱
Instant Telegram Alerts

Real-time threat notifications with screenshot evidence delivered to your phone the moment anomaly is detected.

Intelligence

Dual-model ensemble.
High confidence detection.

Two independent ML models vote on every behavioral window. Consensus required for high-confidence alerts.

Model 1
Isolation Forest

Unsupervised anomaly detection that isolates outliers using random decision trees. Optimal for large behavioral datasets.

O(n log n) complexity — real-time capable
Contamination rate: 5% (tunable)
100 estimators, random_state=42
StandardScaler normalization
Model 2
One-Class SVM

Learns a tight decision boundary around normal behavior using RBF kernel. Superior precision on small datasets.

RBF kernel — handles non-linear patterns
nu=0.05, gamma='scale'
Ensemble with IsolationForest
Unanimous vote = high confidence alert
